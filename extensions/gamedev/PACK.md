# @rune/gamedev

## Skills Included

### threejs-patterns
Three.js patterns — scene setup, camera management, lighting, materials, post-processing, performance optimization, React Three Fiber integration.

### webgl
WebGL patterns — shader programming, GLSL, buffer management, texture handling, instanced rendering, GPU compute.

### game-loops
Game loop architecture — fixed timestep, interpolation, input handling, state machines, entity-component-system (ECS), collision detection.

### physics-engine
Physics integration — Cannon.js, Rapier, Ammo.js. Rigid bodies, constraints, raycasting, collision response, deterministic simulation.

### asset-pipeline
Game asset pipeline — model loading (glTF), texture compression, sprite sheets, audio management, level data formats, asset bundling.
